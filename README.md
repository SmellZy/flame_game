# Used packages

- flame -- game engine package;
- flame_audio -- audio package user for play audion on flame_game;
- flutter_screenutil -- responsive ui package;
- bloc & flutter_bloc -- state manager;
- shared_preferences -- for saving data locally on device;
- flutter_switch -- ready switch widgets;


# How to play

- You nee tap on eggs. To win the level you need to earn required points.

# Egg shop

- You can enter in egg shop by tap on your coin balance widget, on menu of play screens.
